# 🚀 Delta Rev Tech

**Delta Rev Tech** is a futuristic Ed-Tech Android application designed to bridge the gap between physical textbooks and digital learning for students in Grades 1-10.

## 🌟 Overview
In today’s fast-paced educational world, students need more than just static text. **Delta Rev Tech** allows students to scan QR codes directly from their physical books to access:
* Interactive Video Lessons
* Digital Course Content (Maths, Science, English)
* AI-Powered Doubt Clearing

---

## ✨ Key Features
- **Smart QR Scanner:** Integrated with ZXing and Google Code Scanner for rapid URL detection and redirection.
- **AI Chatbot Assistant:** Powered by **Google Gemini AI** to provide 24/7 instant answers to student queries.
- **Secure Authentication:** Robust Google Sign-In and Firebase Auth integration for a personalized user experience.
- **Modern Dashboard:** A sleek, Material 3 UI featuring smooth image loading via Glide.

---

## 🛠 Tech Stack
- **Language:** Java (JDK 11)
- **IDE:** Android Studio
- **SDK:** Target SDK 35 (Android 15)
- **Backend:** Firebase (Auth, Analytics)
- **AI:** Google Generative AI SDK
- **Libraries:** ZXing, Glide, Material Components

---

## 🛡 Security & Production
This project is production-ready. 
- **Code Obfuscation:** ProGuard enabled to protect source logic.
- **Signing:** Signed with a 2,048-bit RSA Keystore for Play Store deployment.
- **Fingerprinting:** Configured with SHA-1 and SHA-256 signatures for Firebase/Google API validation.

---

## 🚀 How to Run
1. Clone this repository: 
   `git clone https://github.com/imurari13/DeltaRevTech.git`
2. Open the project in **Android Studio**.
3. Add your own `google-services.json` to the `/app` folder.
4. Build and Run!
